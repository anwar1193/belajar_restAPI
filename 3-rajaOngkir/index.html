Petunjuk Untuk Pengguna Google Chrome
  1. Pastikan anda sedang menggunakan browser Google Chrome
  2. Klik link berikut : Allow Cors Untuk Ke Halaman Extension
  3. Klik Add to Chrome
  4. Klik Add Extension
  5. Pastikan Extension Allow Cors nya aktif, seperti gambar di bawah ini :
  6. Refresh/Reload halaman aplikasi cek ongkir
  7. Coba kembali untuk menggunakan aplikasi cek ongkir
Petunjuk Untuk Pengguna Mozilla Firefox
  1. Pastikan anda sedang menggunakan browser Mozilla Firefox
  2. Klik link berikut : Allow Cors Untuk Ke Halaman Add Ons
  3. Klik Tambah Ke Firefox
  4. Saat muncul pop up, Klik Tambahkan
  5. Checklist Izinkan Extensi.. , Lalu klik Oke
  6. Pastikan Extension Allow Cors nya aktif, seperti gambar di bawah ini :
  7. Refresh/Reload halaman aplikasi cek ongkir
  8. Coba kembali untuk menggunakan aplikasi cek ongkir
